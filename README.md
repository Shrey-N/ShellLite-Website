# ShellLite Website & Online Compiler

The official source code for the ShellLite website and online compiler.

## Features

- **Online Compiler**: Run ShellLite code directly in the browser.
- **Documentation**: Comprehensive guides and references.
- **Landing Page**: Modern, responsive landing page.

## Quick Start

Run the website locally:

```bash
shl main.shl
```

Then open `http://localhost:8080` in your browser.
